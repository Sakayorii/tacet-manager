# 💊 Tacet Manager

![Open Software License 3](https://img.shields.io/badge/License-OSL%20v3-yellow.svg)

Application to install and use **Tacet** on Android. Built by Sakayori Studio.

## ❓ About

Tacet Manager installs and updates [Tacet](https://github.com/sakayorii/tacet-bundle-next), a client modification for Discord.

## 💪 Features

- 📲 **Install**: Download Discord and install Tacet on it
- 🔁 **Update**: Update Tacet to the latest version
- 🔔 **Notifications**: Be notified when a new version of Tacet is available
- 📃 **Changelog**: See the changelog of Tacet

## 🔽 Download

You can download the most recent version of Tacet Manager from [here](https://github.com/sakayorii/tacet-manager/releases/latest).

## 📚 Everything else

### 🛠️ Building

This is a standard Android (Gradle) project — `./gradlew assembleRelease` produces the APK. Release builds must be signed with your own keystore.
